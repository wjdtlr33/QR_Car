QR 생성 후 "전화하기" 버튼이 실제 번호로 연결됩니다.

GitHub Pages 배포 순서
  1. GitHub 로그인 → wjdtlr33/QR_Car 레포 이동
  2. Add file → Upload files 클릭
  3. 이 index.html 파일 업로드
  4. Settings → Pages 이동
  5. Branch: main / folder: / (root) 선택 후 Save
  6. 약 1~2분 후 주소 생성:
    https://wjdtlr33.github.io/QR_Car
  7. 위 주소를 ③ GitHub Pages 주소에 입력 후 QR 재생성
QR 출력 및 부착 팁
  • PNG 저장 후 5×5cm 이상으로 인쇄 권장
  • 코팅 필름 부착 시 내구성 향상
  • 앞유리 내부 (대시보드 위) 또는 뒷유리 부착
  • 직사광선 반사 주의 — 살짝 각도 기울여 부착
  • 암호화 키는 별도로 안전하게 보관
보안 안내
  • ✅ QR 스캔만으로는 번호 파악 불가
  • ✅ AES-GCM 256bit + PBKDF2 100만회 반복
  • ✅ 서버 없음 — 번호가 외부로 전송되지 않음
  • ⚠️ 브라우저 개발자도구로 소스 분석 시 추출 가능성 있음
